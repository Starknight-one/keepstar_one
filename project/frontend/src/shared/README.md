# Shared

Переиспользуемые модули. Ноль импортов из проекта.

## Папки

- `api/` — HTTP клиент, типы API
- `hooks/` — Общие хуки
- `ui/` — UI компоненты (кнопки, инпуты)
- `logger/` — Логирование
- `lib/` — Утилиты

## Правила

- Нельзя импортировать из `entities/`, `features/`, `app/`
- Только внешние библиотеки и стандартные модули
