# Dev Inspector

Прокси-сервер который инжектит инспектор элементов в страницу. Помогает быстро указать какой элемент нужно поправить.

## Использование

```bash
cd ADW/dev-inspector
npm install
npm start
```

Затем открыть http://localhost:3457 вместо http://localhost:5173

## Хоткеи

- `Ctrl+Shift+K` (или `Cmd+Shift+K` на Mac) — включить/выключить инспектор
- `Esc` — закрыть инспектор
- Клик по элементу — добавить его в список
- Повторный клик — возобновить выбор

## Что показывает

- HTML код элемента (первые строки)
- Номер строки в исходном HTML (если найден)
- Позиция на экране (top/middle/bottom + left/center/right)

## Конфигурация

Переменные окружения:
- `INSPECTOR_PORT` — порт инспектора (по умолчанию 3457)
- `VITE_URL` — URL Vite dev server (по умолчанию http://localhost:5173)
