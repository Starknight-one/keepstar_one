# AI_docs

Документация для агента. Агент может использовать эти документы для понимания контекста и принципов работы.

## Когда использовать

- Агент сам решает искать здесь информацию
- Пользователь явно просит использовать документ
- При работе над архитектурными решениями

## Файлы

- `README.md` — этот индекс
- `ARCHITECTURE_RULES.md` — правила архитектуры проекта (v3.0, обновлён 2026-02-12)
- `Manifesto.md` — манифест продукта, продуктовое видение + что уже построено
- `agent-principles.md` — общие принципы агентного программирования (PITER, leverage points, KPIs)
